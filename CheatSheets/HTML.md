# HTML Cheat Sheet
## Common HTML Elements
all elements go inside diamond brackets <>
- Organization
    - div
    - span
- Headers
    - h1
    - h2
    - h3
    - h4
    - h5
    - h6
- Text
    - p
    - b
    - i
    - strong
    - em
    - pre
    - br
- Groups of Data
    - ol
    - ul
    - li
    - table
    - tr
    - th
    - td
- Miscelanious
    - img src = "url"
    - a href = "url"
## Common HTML Attributes
attributes are key value pairs declared inside the <> of their attached element
- Identifiers
    - id
    - class
    - name
- Visuals
    - hidden
    - style
- Action
    - onclick
